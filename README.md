**Premier League Information** is a modern web application designed to provide fans with a seamless and visually engaging way to access Premier League data. Built with performance and user experience in mind, this application serves as a central dashboard for:

- Checking the latest **League Standings**.
- Viewing upcoming **Match Fixtures**.
- Exploring detailed **Club Profiles**.
- Managing club data (Add/Remove clubs).

The interface features a sleek, dark-themed design inspired by the official Premier League branding, utilizing vibrant gradients and smooth animations.

## ğŸš€ Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/mariosianturi19/premiere-league-app.git
   cd league-frontend
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**

   Navigate to `http://localhost:5173` to view the app.
